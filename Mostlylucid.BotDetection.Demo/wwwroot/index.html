StyloBot Test Site

This is not a real application. It exists solely as a detection target for testing the StyloBot bot detection library.

Use the API endpoints below to fire requests through the full detection pipeline:

  • /_stylobot — Detection dashboard
  • GET /api/demo/open — Unprotected endpoint
  • GET /api/demo/protected — Default policy (block ≥0.7)
  • GET /api/demo/strict — Strict policy (block ≥0.5)
  • GET /api/demo/lenient — Lenient policy (block ≥0.9)
  • GET /api/demo/analyze — Full detection breakdown (JSON)
  • POST /bot-detection/bdf-replay/replay — BDF scenario replay
  • SignatureDemo — Bot signature viewer

BDF scenarios: test-bdf-scenarios/*.json in the repository.
